
# Profile Index

*Format of the  `profiles.ini`  file.*

<br>

## Example

```ini
[General]
StartWithLastProfile=0
Version=2

[InstallS64FU17D]
Default=ky54jb34jlb.Fluff
Locked=1

[Profile0]
Name=Fluff
IsRelative=1
Path=ky54jb34jlb.Fluff
Default=1

[Profile1]
Name=Other
IsRelative=0
Path=/home/fluff/documents/FirefoxProfile
```
